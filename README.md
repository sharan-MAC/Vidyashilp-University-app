## 📖 Overview

This project is developed as part of **B.Tech Computer Science (Data Science Honours)** at **Vidyashilp University**.

It aims to solve inefficiencies in university systems by applying:
- 📊 Data Analysis  
- ⚙️ Optimization Techniques (Linear Programming / MILP)  
- 🧠 Efficient Algorithms  

The system helps in making **data-driven, optimized decisions** for real-world problems.

---

## ❗ Problem Statement

Modern university systems often face:
- Inefficient allocation of resources  
- Scheduling conflicts  
- Lack of intelligent decision systems  

These issues lead to:
- Time wastage  
- Increased costs  
- Reduced operational efficiency
